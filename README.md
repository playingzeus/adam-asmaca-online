# Adam Asmaca Online (Türkçe) — Linkle Davet

## Lokal çalıştırma
1) Node.js 18+ kur
2) Bu klasörde:
```bash
npm install
npm start
```
3) Tarayıcı: http://localhost:3000

Yeni Oda -> Link Kopyala -> Arkadaşına at

## Ücretsiz deploy (Render)
- Render.com -> New -> Web Service
- GitHub'a bu projeyi at (veya zip'i açıp repo yap)
- Build Command: `npm install`
- Start Command: `npm start`
- Environment: Node
- Port: Render otomatik `PORT` verir (server.js bunu okuyor)

Deploy olunca oluşan linki arkadaşınla paylaş.
